### 近期 Bug 修复（3 个回归）

1. **`AttributeError: no attribute '_tray'`** — `__init__` 中的 setup 调用被误缩进到 `_is_recording` setter 内部 (`pass` 之后)，已移回 `__init__`
2. **`FileNotFoundError: project.json`** — `_on_home_record` 和 `_create_project_for_recording` 先设 `_project_dir`（触发 `ProjectSession(value)`→`Project.load`）再写 `project.json`，已调换为先写文件再设属性
3. **"无法在状态 recording 启动录制"** — `_start_recording_from_home` 调用 `start()` 后又 emit `recording_started` 导致 `_on_recording_started` 二次 `start()`，已移除多余信号
