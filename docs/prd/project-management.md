# Recordly 项目管理功能 PRD

## 概述

为 Recordly 桌面录屏工具添加完整的多项目管理功能。用户录制即新建项目，编辑中可手动保存，后续可从项目列表卡片网格中打开继续编辑。

## 用户故事

1. **录制即新建项目**：用户点击录制按钮 → 录制完成 → 自动创建项目（时间戳命名），源视频和缩略图存入项目子目录
2. **项目列表浏览**：主界面显示所有项目的卡片网格（缩略图 + 项目名 + 时长），点击卡片打开项目继续编辑
3. **项目重命名**：双击卡片上的项目名即可重命名
4. **项目删除**：右键卡片菜单删除项目（含所有文件）
5. **工作目录设置**：在设置中配置固定项目目录，启动时自动扫描

## 功能规格

| 决策点 | 结论 |
|--------|------|
| 核心目标 | 完整项目生命周期：录制→编辑→保存→继续编辑 |
| 存储方案 | 目录扫描（固定工作目录） |
| 元数据存储 | Project JSON 内嵌 `name` / `modified_at` / `duration` / `thumbnail_path` |
| 缩略图 | 录制完成自动生成 + 时间线手动覆盖 |
| UI 形态 | 卡片网格（缩略图 + 名称 + 时长） |
| 项目列表操作 | 打开（点击）/ 重命名（双击）/ 删除（右键） |
| 文件结构 | 每个项目一个子目录：`project.json` + `thumbnail.png` + 源视频文件 |
| 新建流程 | 无独立流程 — 录制即新建项目 |

## 涉及模块

| 文件 | 改动 |
|------|------|
| `core/project.py` | 加 `name`、`modified_at`、`duration`、`thumbnail_path` 字段 |
| `core/project_manager.py`（新建） | 目录扫描、创建/删除/重命名/列出项目 |
| `ui/project_card.py`（新建） | 卡片组件（缩略图 + 名称 + 时长） |
| `ui/project_gallery.py`（新建） | 卡片网格布局 + 右键菜单 + 双击重命名 |
| `app/main_window.py` | 替换空 QListWidget 为卡片网格，集成项目操作 |
| `app/settings_dialog.py` | 加"项目目录"设置项 |
| `core/recorder.py` | 录制完成后自动创建项目并生成缩略图 |
